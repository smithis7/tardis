Convert CMFGEN files to TARDIS file formatÂ¶

This script converts CMFGEN input files so that they can be used as input for TARDIS. The script expects the path to the CMFGEN files and the destination path for the TARDIS files as parameters (cmfgen2tardis /path/to/input_file path/to/output/):

$ cmfgen2tardis tardis_example/DDC15_SN_HYDRO_DATA_0.976d tardis_example/
